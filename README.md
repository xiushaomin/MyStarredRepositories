# 我的 Star

由于习惯性看到好的仓库都想Star一下。

但是发现想用的时候可能记不起，所以简单写了小脚本自动生成 README

如果有需要可以跑一下脚本哟！！

```
$ git clone https://github.com/xiushaomin/MyStarredRepositories.git
$ cd to clone folder
$ swift ./MyStarredRepositories.swift
$ 输入Github用户名和密码
$ 等待一会

```
